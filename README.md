### Composer Installation

Add this to to your composer.json file:

```
"repositories": [
  {
    "type": "vcs",
    "url": "git@github.com:flagshipcompany/fpdf"
  }
]
```

then, require it:

```composer require flagshipcompany/fpdf @dev```
